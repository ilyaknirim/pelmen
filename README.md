# Псков: Путешествие Лёхи

Интерактивное образовательное приложение в стиле RPG для Telegram, где пользователь исследует исторические места Пскова вместе с подростком Лёхой.

## Описание проекта

"Псков: Путешествие Лёхи" - это Telegram Mini App, разработанная для подростков 12-16 лет, интересующихся историей. Приложение позволяет исследовать 7 ключевых локаций Пскова, выполнять квесты и собирать исторические артефакты.

## Основные возможности

- Исследование 7 исторических мест Пскова
- Взаимодействие с 5 персонажами
- Выполнение 8+ квестов различной сложности
- Сбор 12+ исторических артефактов
- Интерактивная карта города
- Система достижений
- Сохранение прогресса в облаке Telegram

## Как запустить проект

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/pelmen.git
```

2. Откройте `index.html` в браузере или запустите локальный сервер:
```bash
npx http-server
```

3. Для тестирования в Telegram используйте BotFather для создания бота и получения токена.

## Исторические локации

1. **Псковский Кремль (Кром)** - древняя крепость, основанная в 903 году
2. **Памятник княгине Ольге** - памятник первой женщине-правительнице на Руси
3. **Набережные рек Псковы и Великой** - исторический торговый путь
4. **Довмонтов город** - крепость, построенная князем Довмонтом
5. **Поганкины палаты** - памятник древнерусской архитектуры XVII века
6. **Гремячая башня** - одна из башен Псковского кремля
7. **Псковский историко-художественный музей** - один из старейших музеев России
